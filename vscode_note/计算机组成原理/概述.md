# <center> 概述 <center>

## 1. 相关概念 

1. 摩尔定律：每18个月，集成电路性能翻一番，集成晶体管规模翻一番，价格降一半。

2. 存储程序原理：
   1. 基本思想：计算机要自动完成解题任务，**必须事先将程序和数据以二进制形式存在机器内部**。

   2. 计算机的五大功能：数**据传送、存储、处理功能。操作控制、判断功能**。
        ```
        计算机的两个关键部件：输入设备、输出设备。

        计算机的五个经典部件：输入、输出、存储器、数据通路（运算器）、控制器。
        ```

## 2.  性能指标

1. 机器字长、指令字长、存储字长：
   1. 机器字长（字长）：CPU内部用于整数运算的数据通路的宽度。**一般等于运算器位数和通用寄存器宽度**。

   2. 指令字长：一个指令字中包含的二进制代码的位数。
   3. 存储字长：一个存储单元存储的二进制代码的长度。

2. 吞吐量和响应时间：
   1. 吞吐量：单位时间内处理请求的数量。**系统吞吐量主要取决于主存的存取周期**。

   2. 响应时间：从用户向计算机发送一个请求，到系统对该请求做出响应并获得所需结果的等待时间。
3. 指令执行速度：
   1. MIPS：每秒执行多少百万条指令。

   2. M/G/TFLOPS：每秒执行多少百万/十亿/万亿次浮点运算。
4. Amdahl定律：改进后的执行时间 = 受改进影响的执行时间/改进量 + 不受影响的执行时间。
